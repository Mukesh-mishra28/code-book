What is TypeScript?

TypeScript is open source.
TypeScript strongly typed, object oriented, compiled language developed and maintained by Microsoft, licensed under Apache 2 license.
TypeScript adds additional syntax to JavaScript to support a tighter integration with your editor.
TypeScript can catch errors early in editor.
TypeScript does not run in the browser.
TypeScript is a modern age JavaScript development language.
It is a statically compiled language to write clear and simple JavaScript code.
The code written in typescript is compiled to JavaScript, which then runs in the browser.
TypeScript understands JavaScript and uses type inference to give you great tooling without additional code.

Why TypeScript?

TypeScript simplifies JavaScript code, making it easier to read and debug.
TypeScript gives all the benefits of ES6 (ECMAScript 6).
TypeScript can help avoid painful bugs when writing JavaScript by type checking the code.
TypeScript code is more reliable and easier to refactor.
The type system increases the code quality, readability and makes it easy to maintain and refactor codebase. More importantly, errors can be caught at compile time rather than at runtime.
If you or the majority of the team come from a strongly typed language like C# or Java, and don't want to go all-in on JavaScript, TypeScript is a good alternative.



How to use TypeScript?

TypeScript code is written in a file with .ts extension and then compiled into JavaScript using the TypeScript compiler. A TypeScript file can be written in any code editor. A TypeScript compiler needs to be installed on your platform. Once installed, the command "tsc <filename>.ts" compiles the TypeScript code into a plain JavaScript file. JavaScript files can then be included in the HTML and run on any browser.
